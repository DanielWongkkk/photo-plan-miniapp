## 一、设计概述

### 1.1 产品定位

一款面向摄影初学者的拍照策划工具，核心价值是**高效、专业、易懂**。

### 1.2 设计风格

| 关键词 | 说明 |
|--------|------|
| 简洁 | 减少干扰，聚焦核心功能 |
| 专业 | 摄影相关内容体现专业性 |
| 温暖 | 柔和的色彩，友好的文案 |
| 现代 | 符合当下审美趋势 |

### 1.3 情绪板 (Mood Board)

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   📷 摄影      🌅 黄金时刻      🍃 自然      ✨ 灵感    │
│                                                         │
│   色彩倾向：                                             │
│   • 暖色调为主（橙、金、米白）                           │
│   • 点缀冷色（青、蓝）形成对比                           │
│   • 整体柔和，不刺眼                                     │
│                                                         │
│   视觉参考：                                             │
│   • 小红书（社区感、温暖）                               │
│   • 图虫（专业、简洁）                                   │
│   • iOS 相机界面（直观、高效）                           │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 二、色彩体系

### 2.1 主色

| 名称 | 色值 | 用途 |
|------|------|------|
| 品牌橙 | `#FF8C42` | 主按钮、强调元素、品牌标识 |
| 品牌橙浅 | `#FFB380` | 悬停状态、次级强调 |
| 品牌橙深 | `#E67A35` | 按下状态 |

**使用场景**：
- 主按钮背景
- 选中状态
- 重要图标
- 加载动画

### 2.2 辅助色

| 名称 | 色值 | 用途 |
|------|------|------|
| 暖黄 | `#FFD166` | 提示、警告、高亮 |
| 青绿 | `#06D6A0` | 成功、完成状态 |
| 天蓝 | `#118AB2` | 信息提示、链接 |
| 珊瑚红 | `#EF476F` | 错误、删除 |

### 2.3 中性色

| 名称 | 色值 | 用途 |
|------|------|------|
| 深灰 | `#1A1A2E` | 标题、重要文字 |
| 中灰 | `#4A4A68` | 正文、次要文字 |
| 浅灰 | `#9A9AB0` | 辅助文字、占位符 |
| 超浅灰 | `#F5F5F7` | 背景、分割线 |
| 纯白 | `#FFFFFF` | 卡片背景、输入框 |

### 2.4 渐变色

```
主渐变（用于按钮、卡片）：
┌────────────────────────────────┐
│ #FF8C42 → #FFB380              │
│ 橙色渐变，温暖感               │
└────────────────────────────────┘

背景渐变（用于页面背景）：
┌────────────────────────────────┐
│ #FFF9F5 → #FFFFFF              │
│ 米白到纯白，柔和过渡           │
└────────────────────────────────┘

强调渐变（用于特殊元素）：
┌────────────────────────────────┐
│ #FFD166 → #FF8C42              │
│ 黄橙渐变，活力感               │
└────────────────────────────────┘
```

### 2.5 色彩使用原则

1. **60-30-10 法则**：
   - 60% 中性色（背景、文字）
   - 30% 辅助色（卡片、图标）
   - 10% 主色（按钮、强调）

2. **对比度要求**：
   - 正文文字对比度 ≥ 4.5:1
   - 大标题对比度 ≥ 3:1
   - 符合 WCAG 2.1 AA 标准

## 三、字体规范

### 3.1 字体家族

| 类型 | 字体 | 说明 |
|------|------|------|
| 主字体 | PingFang SC | iOS 系统字体 |
| 备选字体 | SF Pro Text | iOS 英文字体 |
| Android | Noto Sans SC | Android 系统字体 |
| 等宽字体 | SF Mono | 数字、代码 |

### 3.2 字号体系

| 名称 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| 大标题 | 24px | Semibold (600) | 1.3 | 页面标题 |
| 标题 | 20px | Semibold (600) | 1.4 | 卡片标题 |
| 小标题 | 18px | Medium (500) | 1.4 | 区块标题 |
| 正文 | 16px | Regular (400) | 1.5 | 正文内容 |
| 辅助文字 | 14px | Regular (400) | 1.5 | 次要内容 |
| 小字 | 12px | Regular (400) | 1.4 | 标签、时间 |

### 3.3 字重使用

```
Semibold (600) - 标题、重点强调
     ↓
Medium (500) - 小标题、按钮文字
     ↓
Regular (400) - 正文、一般内容
```

## 六、图标规范

### 6.1 图标风格

- **风格**：线性图标（Outlined）
- **粗细**：1.5px
- **圆角**：适度圆角，柔和感
- **大小**：主要使用 24px

### 6.2 图标库

| 图标 | 名称 | 用途 |
|------|------|------|
| 📷 | camera | 拍摄、设备 |
| 📍 | location | 地点 |
| 📅 | calendar | 日期 |
| 🕐 | clock | 时间 |
| 🌤️ | weather | 天气 |
| ✨ | sparkles | 生成、AI |
| 🔄 | refresh | 重新生成 |
| ⚙️ | settings | 设置 |
| 👤 | user | 用户、个人中心 |
| 📤 | share | 分享 |
| 🗑️ | delete | 删除 |
| ➕ | plus | 添加 |
| ▶️ | play | 展开、播放 |
| ✓ | check | 完成、选中 |

### 6.3 图标使用规范

```
正确 ✓：
- 图标与文字对齐居中
- 图标颜色与文字一致或略浅
- 图标大小与文字行高协调

错误 ✗：
- 图标过小难以辨认
- 图标颜色过于突兀
- 图标与文字间距不当
```

## 七、动效规范

### 7.1 过渡动画

| 场景 | 动画 | 时长 | 缓动函数 |
|------|------|------|----------|
| 页面切换 | 滑入/滑出 | 300ms | ease-out |
| 弹窗出现 | 缩放 + 淡入 | 200ms | ease-out |
| 按钮点击 | 缩小 0.98 | 100ms | ease |
| 加载动画 | 旋转 | 1000ms | linear (循环) |

### 7.2 微交互

#### 按钮点击
```
默认状态 → 按下（缩小 2%）→ 松开（恢复）
         100ms         100ms
```

#### 标签选择
```
未选中 → 点击 → 选中（背景色变化 + 轻微缩放）
              150ms
```

#### 卡片展开
```
折叠 → 点击 → 展开（高度变化 + 内容淡入）
            300ms
```

### 7.3 加载动画

#### 生成策划动画
```
帧 1    帧 2    帧 3    帧 4
  ◠       ◡       ◠       ◡
  
配合文字：
"分析拍摄场景..."
"匹配设备参数..."
"搜索参考样片..."
"生成完成！"
```

## 十、响应式适配

### 10.1 设计基准

- **基准宽度**：375px（iPhone SE / iPhone 14）
- **最大宽度**：428px（iPhone 14 Pro Max）
- **最小宽度**：320px（旧机型兼容）

### 10.2 适配原则

| 元素 | 适配方式 |
|------|----------|
| 文字 | 固定字号，不缩放 |
| 图标 | 固定尺寸，不缩放 |
| 间距 | 百分比 + 最小值限制 |
| 卡片 | 宽度 100% - 边距 |
| 图片 | 宽度自适应，高度按比例 |

### 10.3 横屏处理

- 不支持横屏（强制竖屏）
- 理由：拍照策划场景下用户多为竖屏使用
