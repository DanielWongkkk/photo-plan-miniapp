## 二、页面交互详情

### 2.1 登录页 (login.html)

#### 页面目标
- 完成用户登录/注册
- 获取微信授权
- 引导用户设置头像昵称

#### 交互元素

| 元素 | 交互类型 | 触发行为 |
|------|----------|----------|
| 微信一键登录按钮 | 点击 | 调用 `wx.login()` 获取授权 |
| 头像上传区域 | 点击 | 打开图片选择器 |
| 昵称输入框 | 输入 | 实时验证（最少2字符） |
| 完成登录按钮 | 点击 | 提交登录信息，跳转首页 |
| 用户协议链接 | 点击 | 弹出协议详情 |

#### 状态说明

```
初始状态：
- 微信登录按钮可点击
- 头像区域显示占位符
- 昵称输入框为空
- 完成登录按钮禁用（灰色）

头像已选择：
- 头像区域显示选择的图片
- 添加边框高亮效果

昵称已输入（≥2字符）：
- 完成登录按钮激活（橙色渐变）

全部完成：
- 完成登录按钮可点击
- 点击后跳转首页
```

#### 异常处理

| 场景 | 处理方式 |
|------|----------|
| 网络错误 | Toast 提示 "网络异常，请重试" |
| 微信授权失败 | Toast 提示 "授权失败，请重试" |
| 昵称重复 | Toast 提示 "昵称已被使用" |

## 三、通用交互规范

### 3.1 手势操作

| 手势 | 场景 | 行为 |
|------|------|------|
| 点击 | 按钮、卡片、列表项 | 触发对应操作 |
| 长按 | 图片 | 弹出菜单（保存、分享） |
| 下拉 | 列表页 | 刷新内容 |
| 左滑 | 历史记录项 | 显示删除按钮 |
| 双击 | 图片 | 放大/缩小 |
| 捏合 | 图片 | 缩放操作 |

### 3.2 反馈机制

| 场景 | 反馈类型 | 样式 |
|------|----------|------|
| 按钮点击 | 视觉 + 触觉 | 按钮缩小 2% → 恢复 + 轻微震动 |
| 操作成功 | Toast | 深色圆角条，顶部居中，2秒消失 |
| 操作失败 | Toast | 深色圆角条，顶部居中，3秒消失 |
| 加载中 | 骨架屏/进度条 | 灰色占位 + 动画扫过效果 |
| 网络错误 | 弹窗 | 居中弹窗，提示重试 |

### 3.3 Toast 规范

```
位置：页面顶部，距离状态栏 24px
样式：
┌─────────────────────────────┐
│  ✓ 保存成功                 │
└─────────────────────────────┘
背景：rgba(26, 26, 46, 0.9)
文字：白色，14px，居中
圆角：20px
动画：从上滑入 → 停留 2s → 滑出消失
```

### 3.4 确认弹窗规范

```
位置：屏幕居中
样式：
┌─────────────────────────────────────┐
│                                     │
│           确认删除？                │
│                                     │
│   删除后无法恢复，确定删除吗？       │
│                                     │
│  ┌──────────┐  ┌────────────────┐  │
│  │   取消   │  │     删除       │  │
│  └──────────┘  └────────────────┘  │
│                                     │
└─────────────────────────────────────┘
宽度：屏幕 80%（最大 320px）
背景：白色，圆角 16px
遮罩：黑色 50% 透明度
取消按钮：白底灰边
确认按钮：品牌色/危险色
```


```
场景：AI 服务不可用或生成失败

处理：

3. 记录日志供排查

┌─────────────────────────────────────┐

│                                     │
└─────────────────────────────────────┘
```

```

3. 标注来源链接，用户可手动查看

样式：
┌─────────────┐
│             │
│   🖼️        │  ← 占位图
│   加载失败   │
│             │

```
