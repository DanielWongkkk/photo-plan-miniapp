# 📷 拍照策划小程序

> 一款面向摄影初学者的拍照策划工具，AI 智能生成拍摄方案

## 项目简介

刚学摄影，发现拍照策划很重要，但懒得写？这个小程序帮你一键生成拍照策划方案：

- 🎯 **智能策划**：输入主题、地点、设备，AI 自动生成拍摄方案
- 📸 **样片参考**：AI 搜索真实样片，知道成片效果
- ⚙️ **设备适配**：基于你的实际设备生成参数建议
- 💡 **省时省力**：不用手动搜索整理，自动生成可落地的策划

## 功能特性

### 核心功能
- ✅ 微信一键登录
- ✅ 设备管理（支持多套设备）
- ✅ AI 生成拍照策划
- ✅ 拍摄清单（Shot List）
- ✅ 样片参考
- ✅ 参数建议
- ✅ 历史记录

### 输出内容
生成的拍照策划包含：
- 📍 地点建议（最佳取景点）
- 🕐 时间建议（光线分析）
- 🌤️ 天气影响提示
- 📷 设备配置建议（镜头、参数）
- 📋 拍摄清单（景别、构图、样片）

## 技术栈

### 前端（微信小程序）
- 原生小程序框架
- ES6+ JavaScript
- WXSS 样式

### 后端
- Node.js + Express
- MongoDB
- 多 AI 模型支持（通义千问、DeepSeek、GLM 等）

## 快速开始

### 小程序
1. 导入 `miniprogram` 目录到微信开发者工具
2. 修改 `app.js` 中的 API 地址
3. 编译运行

### 后端服务
```bash
cd server
npm install
cp .env.example .env
# 编辑 .env 配置
npm start
```
