# 📷 拍照策划小程序

> 版本：v0.2.0  
> 更新日期：2026-03-16

一款面向摄影初学者的拍照策划工具，AI 智能生成可落地的拍摄方案。

## 项目简介

刚学摄影，发现拍照策划很重要，但懒得写？这个小程序帮你一键生成拍照策划方案：

- 🎯 **智能策划**：输入主题、地点、设备，AI 自动生成拍摄方案
- 📍 **具体取景点**：带定位导航，直接到点拍摄
- 📸 **样片参考**：每个点位关联真实样片，标注来源
- 🌤️ **天气驱动**：基于天气预报推荐最佳拍摄日期
- 💃 **摆姿引导**：人像拍摄提供详细动作指导
- ⚙️ **设备适配**：基于你的实际设备生成参数建议

## 功能特性

### 核心功能
- ✅ 微信一键登录
- ✅ 设备管理（支持多套设备）
- ✅ AI 生成拍照策划
- ✅ 天气预报 + 日期推荐
- ✅ 具体取景点（带导航）
- ✅ 样片参考（标注来源）
- ✅ 人像摆姿引导
- ✅ 拍摄清单打卡
- ✅ 参数配置建议
- ✅ 后期调色指南

### 策划输出内容

生成的拍照策划包含：

| 模块 | 内容 |
|------|------|
| 🗓️ 日期推荐 | 未来5天天气预报 + 拍摄适宜度评分 |
| 🕐 时间推荐 | 晨光/黄金时刻/蓝调时刻分析 |
| 📍 取景点 | 具体位置 + 导航 + 最佳时间 + 参数 |
| 📷 样片 | 每个点位的参考样片（标注来源） |
| 💃 摆姿 | 人像拍摄的动作引导 |
| ⚙️ 参数 | 光圈/快门/ISO/白平衡建议 |
| 📝 清单 | 可打卡的拍摄项目列表 |
| 🎨 后期 | 调色步骤 + 预设推荐 |
| ⚠️ 注意 | 地点相关注意事项 |

## 技术栈

### 前端（微信小程序）
- 原生小程序框架
- ES6+ JavaScript
- WXSS 样式

### 后端
- Node.js + Express
- Open-Meteo 天气 API（免费）
- 多 AI 模型支持（通义千问、DeepSeek、GLM 等）

## 快速开始

### 小程序
1. 导入 `miniprogram` 目录到微信开发者工具
2. 修改 `utils/api.js` 中的 API 地址
3. 编译运行

### 后端服务
```bash
cd server
npm install
cp .env.example .env
# 编辑 .env 配置 AI API Key
npm start
```

## 更新历史

### [v0.2.0] - 2026-03-16

#### 新增功能
- **天气驱动的日期推荐**：集成 Open-Meteo API，获取未来5天天气预报，计算拍摄适宜度评分
